## 📌 About The Project

**SimulVidere** is a real-time YouTube watch party web app that lets two or more people — sitting anywhere in the world — watch the same YouTube video together, perfectly in sync.

No downloads. No sign-ups. Just share a room code and watch together. 🍿

> 💡 *"Simul Videre"* is Latin for **"To See Together"**

---

## 🌟 Key Features

| Feature | Description |
|--------|-------------|
| 🏠 **Create & Join Rooms** | Host creates a room, shares a 4-digit code, guest joins instantly |
| ▶️ **Synchronized Playback** | Play, Pause, and Seek synced for all viewers in real time |
| 💬 **Live Chat** | Chat while watching — messages appear instantly for everyone |
| ❤️ **Emoji Reactions** | React with ❤️ 🔥 😂 😮 👏 🍿 — emojis float across the screen |
| 😄 **Join Notification** | Host gets a happy face notification + popcorn effect when friend joins |
| 👥 **Live Viewer Count** | See how many people are watching live |
| 🎬 **Cinematic Intro** | Netflix-style animated intro screen on launch |
| 📱 **Responsive Design** | Works on desktop and mobile browsers |

## 🛠️ Tech Stack

```
Frontend          →  HTML5, CSS3, Vanilla JavaScript
Real-time Sync    →  Firebase Realtime Database
Video Player      →  YouTube IFrame Player API
Animations        →  CSS Keyframes & Transitions
Fonts             →  Google Fonts (Bebas Neue, Inter)
Deployment        →  Netlify
Version Control   →  Git & GitHub
```

### Detailed Breakdown

- **HTML5** — Semantic structure, single-page layout with multiple screens
- **CSS3** — Custom properties (variables), keyframe animations, flexbox layout, responsive design
- **Vanilla JavaScript (ES6+)** — DOM manipulation, async/await, real-time event listeners
- **Firebase Realtime Database** — NoSQL cloud database for live sync of video state, chat, reactions, and member presence
- **YouTube IFrame Player API** — Programmatic control of video playback (play, pause, seek)
- **Netlify** — Static site hosting with continuous deployment from GitHub

## 🎮 How To Use

```
1. Open the site
       ↓
2. Enter your name → Click "Room Banao"
       ↓
3. Share the Room Code (e.g. YT-4821) with your friend
       ↓
4. Friend opens site → enters name + code → Joins
       ↓
5. Paste any YouTube link → Click Load
       ↓
6. Watch together in sync! React, chat, enjoy 🎬
```
